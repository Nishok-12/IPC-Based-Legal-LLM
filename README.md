# IPC-Based Legal LLM using RAG

An AI-powered Legal Information System that provides accurate and context-aware legal information from the Indian Penal Code (IPC) using Retrieval-Augmented Generation (RAG).

---

## Features

- Retrieval-Augmented Generation (RAG)
- Semantic Legal Search
- IPC-Based Legal Query Handling
- Context-Aware Legal Response Generation
- Reduced Hallucination using Document Grounding
- Locally Hosted LLaMA Models using Ollama
- Interactive Streamlit-Based Interface

---

## Technologies Used

- Python
- FAISS
- Sentence Transformers
- LLaMA
- Ollama
- Streamlit
- LangChain
- Natural Language Processing (NLP)

---

## System Workflow

1. IPC legal documents are preprocessed
2. Legal text is split into chunks
3. Sentence Transformers generate embeddings
4. FAISS performs semantic similarity search
5. Relevant IPC sections are retrieved
6. LLaMA generates context-aware legal responses
7. Responses are displayed through Streamlit

---

## Applications

- Legal Information Retrieval
- IPC Section Identification
- Legal Query Assistance
- Educational Legal Support
- AI-Based Legal Research
